## Installation

Clone and move to the repository

# important

don't forget to setup your mysql and .env

```bash
git clone https://github.com/AaronFabian/motogp-streams.git;
cd motogp-streams;
```

# install packages

```bash
npm i;
```

# run the server

```bash
npm run dev;
```
